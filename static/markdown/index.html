Это инструкция о том, как форматировать новости.

Чтобы добавить заголовок, можно изпользовать такие выражения:

# Заголовок
## Поменьше
### Ещё меньше

Результат:

Заголовок

Поменьше

Ещё меньше

Чтобы добавить ссылку:

[Google](google.com)

Результат:

Google

Чтобы выделить текст:

Можно делать *курсивный* и **жирный** шрифт

Результат:

Можно делать курсивный и жирный шрифт

Списки:

1. Начало
2. Середина
3. Конец

Результат:

  1. Начало
  2. Середина
  3. Конец

Или:

* Просто
* список
* с точками

И получится:

  • Просто
  • список
  • с точками

Мало когда нужно, но полезно знать:

Чтобы зачеркнуть текст:

Зачеркнём ~~текст~~

Результат:

Зачеркнём текст

Цитата

> Я помню чудное мгновенье...

Результат:

Я помню чудное мгновенье...
